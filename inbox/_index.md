# Inbox

> **Staging area for everything that doesn't yet have a clear place in the system.**

## Purpose

Everything unclear, just coming in, or that still needs to be classified lands here:
- Raw thoughts, ideas, brainwaves
- Articles, PDFs, links, screenshots for later processing
- Messages that might be relevant
- Everything you're not sure where it belongs

## Workflow

1. Drop the content here (file, link, short note).
2. On the next review, decide where the content belongs (see below).
3. **Move** from the inbox, don't copy — the inbox is transient.

## Target locations (routing)

| Content type | Target |
|---|---|
| Important person | `people/<name>.md` |
| Company / client / supplier | `organisations/<name>.md` |
| Active project | `projects/<project>/_index.md` |
| Meeting / call | `interactions/<date>-<topic>.md` |
| Source / article / book | `knowledge/raw/<source>.md` |
| Curated knowledge | `knowledge/wiki/<topic>.md` |
| Task | `ops/todo.md` |
| Decision | `ops/decisions.md` |
| New rule | `RULES.md` |
| Repeatable process | `skills/<skill>.md` |
| Trash / not relevant | `trash/` |

## Rules

- The inbox is **transient**. Contents here should be classified regularly.
- Don't let content grow here for more than 2 weeks without classification.
- Never use inbox content as a source for answers without classifying it first.
