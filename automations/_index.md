# Automations

> **Outputs from external automations.**

## Purpose

Files here are generated by external tools (webhooks, scheduled scripts, IFTTT, Zapier, n8n, etc.). They are **read-only from the wiki's perspective** — the wiki doesn't write here manually.

## Structure

```
automations/
├── _index.md
└── <source>/
    └── <YYYY-MM-DD>-<event>.md
```

## Rules

- Automation outputs land here first — classify to the right area when reviewed.
- Don't manually edit automation outputs (they may be overwritten by re-runs).
- Interesting information from automation outputs → classify to `knowledge/raw/` or `inbox/`.
