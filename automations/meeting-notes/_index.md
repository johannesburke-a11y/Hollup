# Meeting Notes — Automation Output

> Auto-captured meeting notes from Google Calendar + Google Drive.
> Generated by: `scripts/sync-meeting-notes.ps1` → `scripts/google-calendar/sync-meeting-notes.js`
> Schedule: runs automatically via Windows Task Scheduler (every 30 min during working hours)

---

## What lands here

- One file per ended meeting that had a Google Meet link
- Files are named `YYYY-MM-DD-<meeting-slug>.md`
- Each file contains: meeting meta, attendees, Meet link, Google Doc notes (if found)

## What to do with these files

1. **Quick review** — scan after each meeting block
2. **Promote to `interactions/`** — if the meeting needs follow-up tracking
3. **Extract tasks** — add action items to `ops/todo.md`
4. **Leave as-is** — if the meeting was routine and needs no further action

## Rules

- Don't manually edit these files — they may be overwritten on re-runs
- Interesting insights → classify to `knowledge/` or `inbox/`
- Action items → always go to `ops/todo.md`

## Sync log

See `sync.log` in this folder for run history.
