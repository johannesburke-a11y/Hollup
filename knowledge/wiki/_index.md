# Knowledge / Wiki

> **Curated, classified, linked knowledge.**

## Purpose

This is where raw sources become real knowledge. Each page represents one concept, pattern, or topic — written in your own words, linked to sources and related pages.

## File naming

```
knowledge/wiki/<topic-slug>.md
```

## Frontmatter schema

```yaml
---
type: knowledge
status: active          # active | outdated | draft
tags: [topic/<...>, area/<...>]
---
```

## Rules

- Wiki pages are written in **your own words** — not copy-pasted from sources.
- Each page links to its raw sources in `knowledge/raw/`.
- When a page becomes outdated: mark it (`status: outdated`) and update — don't maintain in parallel.
