# Interactions

> **Meetings, calls, conversations — with context.**

## Purpose

Every significant interaction that produces a decision, action item, or important context gets its own file.

## File naming

```
interactions/YYYY-MM-DD-<topic-slug>.md
```

Example: `interactions/2026-06-15-project-kickoff.md`

## When to create an interaction file?

- When decisions were made.
- When action items emerged.
- When important context was shared.
- Not for: casual small talk, trivial status pings.

## Rules

- Interaction files are **append-only** after creation — don't rewrite what was said.
- Action items from interactions → also in `ops/todo.md` (central!).
- Use the template: `resources/templates/interaction.md`.
