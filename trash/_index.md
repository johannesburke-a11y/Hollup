# Trash

> **Reviewable recycle bin.**

## Purpose

Files that are probably no longer needed, but haven't been permanently deleted yet. Review periodically — if still not relevant after 30+ days, delete permanently.

## Rules

- Move files here instead of deleting directly (see `RULES.md §9`).
- Add a short note at the top of each trashed file: why was it moved here?
- Review trash at least monthly.
- Permanent deletion only after review.
